# Shot List & Visual Prompts

Use these prompts in your image/video generator or Canva backgrounds; keep a consistent dark + teal/gold theme.

1) **Title card**: “Web3 AI Agents — Multi-Agent Trading Copilot”. Prompt: “Sleek dark trading dashboard, teal/gold accents, floating panels with crypto charts and agent icons, vector style, professional.”

2) **Problem slide**: Overlay bullets: “Data fragmented”, “Cognitive overload”, “Risk blind spots”. Background prompt: “Chaotic collage of charts/news/windows, dimmed, with subtle warning icons.”

3) **Why agents**: Hub-and-spoke graphic: Orchestrator center, nodes: Search, DataEng, Analytics, Risk, Trading. Prompt: “Vector network of six nodes labeled roles, dark theme, clean lines.”

4) **Architecture**: Diagram showing Orchestrator -> five agents; callouts: Gemini, A2A, MCP, Memory, Human-in-loop. Prompt: “System architecture schematic, boxes and arrows, dark UI blueprint, teal/gold highlights.”

5) **Demo snippet**: Screen capture of terminal command `python -m agents.run_orchestrator --request "analyze ETH, SOL; risk=balanced"` and a small markdown report excerpt (ideas/metrics/risk/plan). If recording, keep it ~10s clip.

6) **Human-in-loop**: Short overlay: “Pause for approval when risky trades”. Prompt: “Notification card with approve/reject buttons, overlaid on trading UI, minimal.”

7) **Build stack**: Icons/text: Python, google-adk, Gemini, A2A, MCP, Binance public data (optional), Memory/compaction, Evaluator Agent, Human-in-loop. Prompt: “Tech stack icons on dark board, neat grid, teal highlights.”

8) **Closing card**: “Faster insights, safer trades. Next: richer data, broader evals, Agent Engine demo.” Prompt similar to title card.

Optional real footage:  
- 10s terminal run; 5–8s scroll of generated report; 5s showing approval/pending message (if AUTO_APPROVE_TRADES=0).
